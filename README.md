# Data Professional Survey Dashboard

## Overview
This project is an interactive Power BI dashboard built using a survey dataset of data professionals. The dashboard provides insights into salary trends, favorite programming languages, work-life balance, salary satisfaction, and the challenges faced while entering the data industry.

## Objectives
- Analyze salary distribution across different job roles.
- Identify popular programming languages among data professionals.
- Understand work-life balance and salary satisfaction levels.
- Explore career transition challenges in the data industry.
- Present insights through interactive visualizations.

## Tools Used
- Power BI
- Power Query
- Microsoft Excel

## Data Preparation
- Imported data from Excel.
- Cleaned and transformed data using Power Query.
- Handled data formatting and prepared the dataset for analysis.
- Created visualizations and dashboard components.

## Key Insights
- Salary comparison across different data-related job roles.
- Most preferred programming languages among professionals.
- Respondent distribution by country.
- Work-life balance and salary satisfaction analysis.
- Difficulty level of breaking into the data industry.
